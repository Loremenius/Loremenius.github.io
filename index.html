Family Wishlist Github Repo: Back-end

Month long project created by myself. A recreation of a website my father and his friend made back in the early 2000's. This website is for distant family to post what items they would like but dont have the money to spend it on. People can look at the list, buy the item, and mark the item as purchased and give the person the item as a gift. It features a Front end with react, css styling, and redux for storage management and a back end using Node, express, and JSON web tokens.
